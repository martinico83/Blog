INTRODUCCION A SELECTORES CSS


Selector de elemento

p {
color: blue;
}


Selector de clase

.cliente {
color: blue;
}

-Una clase se puede crear multiples veces (es reutilizable) e inicia con un punto.


Selector de ID

#cliente {
color: blue;
}

-Puedes tener multiples ID`s por pagina pero sus nombres no pueden repetirse.


Selector de Atributo

[src="logo.jpg"] {
color: blue;
}

-Selecciona elementos basado en algun atributo que tenga.


Combinacion de descendentes

.cliente .nombre {
color: blue;
}

-Selecciona los elementos hijos cuyos padre sea una clase (o ID) es especifico.


Todos los hijos

.cliente > p {
color: blue;
}

-Aplica a todos los hijos.

ESPECIFICIDAD EN CSS


¿Que valores del CSS seran tomados en cuenta por el navegador?


¿Que es la especificidad?

Es como el navegador va a mostrar el CSS de acuerdo a que tan especifico es el selector que hemos creado.

CSS es en cascada, eso no significa que si un selector aparece despues sera tomado en cuenta sino mas bien su especificidad

-como ultima opcion puedes usar la palabra "important" pero no es recomendable

p {
color: blue!important;
}

FLEXBOX

-Flexbox fue diseñado como un modelo unidimensional para crear layouts

Los dos ejes de Flexbox:

En Flexbox solo puedes colocar y distribuir tus elementos en una dirección fila (row) o columna (column). Row es aplicado por default al definir un display: flex; Los otros valores son row-reverse, column y column-reverse.
Si elegimos row o row-reverse os elementos hijos se colocarán de izquierda a derecha uno junto al otro.
Al elegir column o column-reverse los elementos se colocarán de arriba hacia abajo.
Flexbox es especialmente diseñado para alinear elementos en tus diseños.
No añade efectos de animación, ni textos, es una tecnología utilizada únicamente para los layouts y sustituye a los floats o table-cell

¿Que es CSS Box Model?

-El tamaño de lo que se muestra en pantalla esta delimiado por 4 cosas:
* Tamaño de contenido.
* Tamaño de relleno (Padding).
Tamaño del borde y el margen.

-Un truco que se puede utilizar para que al momento de darle un width a tu foto, esta no le afecte el padding o el border, es usar este codigo.

html {
font-size: 62.5%;
box-sizing: border-box;
}
*,
*:before,
box-sizing: inherit;
}
